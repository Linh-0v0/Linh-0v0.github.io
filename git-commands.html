Basic Git commands

December 4, 2020 / by Vu Bui Khanh Linh

Configure Github credentials

git config ––global user.name “github_username”
git config ––global user.email “email_address”

Replace github_username and email_address with your GitHub credentials that you have registered.


Clone a Github repository (Note: create/have a repo before clone)

git clone repository_url


To update the latest version of the repo (when you already have the data cloned from the repo)

git pull


Add a new file -> Commit -> Push

git add .
git commit –m “message_(name_for_the_commit)”
git push


To check which file is uncommited (unrecorded)

git status


Further Git-branch-commands will be in Git Branching in a nutshell.

Merge your branch to the main branch:

git merge


Show the file differences which aren’t yet staged (committed)

git diff
git diff [fist_branch] [second_branch]
