Side Projects

Particeps is the group project of the Sketchers, voted by all members. However, other members' projects are equally great and are worth a quick read

Linh's Project involves creating an application that will show the user the nearest public sanitary stations (urinals, mobile toilets, etc.). This application, if realistically applied, can reduced the pollution of the already heavily polluted city of Sai Gon.

Thao's Project

Thao's project includes creating a gps-based application that is more localized to Sai Gon, which is capable of showing places in alleys that often do not show up on large scale applications. This way, tourists and travellers can enjoy an authentic Vietnamese experience when they visit Sai Gon.

Sourabi's Project

Sourabi's Project involves creating an application to to improve farmers' crops. Basically, the app
tell the users about basic scientific info like dirt types, nutrional values, plants types etc...
